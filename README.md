# js-scroll_puncture

## Installation

You can install the library as a npm module by running the following command:
```
npm install js-scroll_puncture
```

## Usage

```javascript
import {clickPop,closePop} from 'js-scroll_puncture'

function openModal(){
  clickPop()
  //do something
}

function closeModal(){
  closePop()
  //do something
}

```
